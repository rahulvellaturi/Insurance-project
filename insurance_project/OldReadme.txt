## 🎯 **What is AssureMe?**

AssureMe is a **production-ready, enterprise-grade insurance client management system** featuring:

- ✅ **React 18.3.1** with functional components and modern hooks
- ✅ **TypeScript** with 98% type coverage for type safety
- ✅ **Redux Toolkit** for state management with persistence
- ✅ **Node.js + Express** backend with PostgreSQL database
- ✅ **Comprehensive Testing** with 90%+ coverage (2,000+ test cases)
- ✅ **Enterprise Security** with JWT, MFA, and validation
- ✅ **Production Deployment** ready with free hosting options

## ⚡ **Quick Start**

```bash
# Clone and setup everything automatically
git clone https://github.com/rahulvellaturi/AssureMe_Insurance
cd AssureMe_Insurance
chmod +x setup.sh
./setup.sh

# Start development
./start.sh
```

**Access your app:**
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
- **API Docs**: http://localhost:5000/api-docs

**Test credentials:**
- **Client**: `john.doe@email.com` / `password123`
- **Admin**: `admin@assureme.com` / `admin123`
